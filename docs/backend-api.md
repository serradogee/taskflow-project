# Herramientas y librerías de backend/frontend

## Axios
Cliente HTTP moderno para hacer peticiones asíncronas desde el frontend. Alternativa a fetch, más fácil de usar en proyectos grandes.

## Postman
Herramienta para probar y documentar APIs. Permite enviar GET, POST, DELETE, etc., con distintos cuerpos y headers.

## Sentry
Servicio de monitoreo de errores y rendimiento. Detecta fallos en producción y genera alertas automáticas.

## Swagger
Herramienta para documentar APIs REST. Permite generar documentación interactiva para que otros desarrolladores vean los endpoints y prueben la API.

## Uso
- Axios/fetch → comunicar frontend con backend
- Postman → probar API manualmente
- Sentry → monitorear errores reales
- Swagger → documentar API y facilitar integración

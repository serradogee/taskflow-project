# Primer contacto con Cursor

En este documento se describe la experiencia de uso de Cursor con el proyecto TaskFlow, incluyendo exploración de la interfaz, autocompletado, chat contextual, edición inline y Composer.

---

## 1. Instalación y apertura del proyecto

- Se instaló Cursor siguiendo las instrucciones oficiales.  
- Se abrió el proyecto **TaskFlow** dentro de Cursor, cargando automáticamente los archivos HTML, CSS y JS.

---

## 2. Exploración de la interfaz

Elementos explorados:

- **Explorador de archivos:** Permite navegar rápidamente entre todos los archivos del proyecto.  
- **Terminal integrada:** Se puede ejecutar `npm`, `git` y otros comandos directamente desde Cursor.  
- **Chat contextual:** Permite hacer preguntas sobre el código actual y recibir explicaciones.  
- **Herramientas de edición:** Incluyen autocompletado, edición inline y Composer para modificar varias partes del proyecto simultáneamente.

# Comparación de asistentes de IA: ChatGPT vs Claude

## 1. Explicación de conceptos técnicos

Se pidieron a ambos asistentes que expliquen tres conceptos de JavaScript: **closures**, **event loop** y **hoisting**.

### ChatGPT

- **Closures:** Explica que son funciones que recuerdan el entorno donde fueron creadas, incluyendo variables externas. Da un ejemplo con función que retorna otra función.
- **Event Loop:** Describe cómo JavaScript maneja la cola de tareas y callbacks, mencionando el stack de ejecución y la cola de eventos.
- **Hoisting:** Explica que las declaraciones de variables y funciones se elevan al inicio del scope, con ejemplos usando `var` y `let`.

### Claude

- **Closures:** También explica la captura de variables externas y el alcance léxico, con ejemplos claros de uso en funciones anidadas.
- **Event Loop:** Describe la ejecución de tareas asíncronas y microtasks/promises, con ejemplos más visuales de flujo.
- **Hoisting:** Presenta ejemplos de elevación de funciones y variables, con advertencias sobre diferencias entre `var`, `let` y `const`.

**Comparación de claridad y profundidad:**  

| Concepto      | ChatGPT | Claude |
|---------------|---------|--------|
| Closures      | Claro, ejemplo sencillo | Muy claro, ejemplo visual |
| Event Loop    | Detallado, explica stack y cola | Más didáctico, visual del flujo |
| Hoisting      | Preciso, ejemplos básicos | Explica diferencias entre var/let/const |
